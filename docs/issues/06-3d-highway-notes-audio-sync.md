# 06 — 3D 高速公路 + 音符飛行 + billboard + 音同步

> 來源:docs/PRD.md

## What to build

消費 `TypingChart` 建立 Three.js 3D 場景:音符從遠方朝鏡頭前的判定平面飛來,字母永遠 billboard 面向鏡頭保持可讀。左手的道在畫面左側、右手在右側(呼應顏色=手)。飛行時間為我方固定可調常數(預設約 1.5–2s),**不沿用譜面 NJS**;音符節奏落點仍嚴格照 `tSec`。以 `AudioContext.currentTime` 為主時鐘,音符移動與音樂同步;提供手動校準 offset(同時影響視覺,判定於後續 issue)。本切片純視覺+音,無判定。

## Acceptance criteria

- [ ] 載入譜面後進入 3D 畫面,音符隨音樂從遠方飛向判定平面
- [ ] 字母始終正面朝向鏡頭、清晰可讀
- [ ] 左右手音符分別出現在畫面左/右側
- [ ] 音符抵達判定平面的時間與音樂節拍一致(對拍)
- [ ] 飛行時間可調且各歌一致;調整 offset 會整體位移音符抵達時機
- [ ] 效能在範例譜面下順暢(無明顯掉幀)
