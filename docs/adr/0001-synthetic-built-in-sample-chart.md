# 內建範例譜面採用自製合成資產,不用真實 BeatSaver 譜面

issue 01 的 tracer bullet 需要一首內建範例譜面(info.dat + v2 難度檔 + song.egg)。我們決定**手寫一份極小的 v2 譜面 + 自行合成的簡單旋律 OGG**,而非把現成的社群譜面塞進 repo。

理由:(1) 版權乾淨——音樂與譜面資料皆為自製,repo 日後可公開無虞;(2) 可決定性——手寫譜面每顆音符的 beat/color/col/row 皆已知,能對 `compileChart` 逐一斷言輸出,符合「純函式 + fixtures」的測試慣例;(3) 對拍可驗證——合成音訊的旋律音可精準落在譜面 beat 上,肉眼/耳朵即可驗證按鍵時間點與音訊同步。

代價:範例不具娛樂性;真實譜面體驗留待 issue 04(zip 拖放)。
